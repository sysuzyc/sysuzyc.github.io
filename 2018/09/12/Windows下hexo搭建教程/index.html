使用Hexo提交文档

我们可以使用一些基本的指令来进行本地文件的提交：

hexo new "new title"：生成一个新的md文档，按自己需要命名

hexo clean ：删除旧的public文件

hexo generate 或 hexo g：生成新的public文件

hexo deploye 或 hexo d：开始部署到GitHub之中
