发表于 2020-03-14 | 分类于 LeetCode ， 难题

通过投票对团队排名

排序 vector

题目

思路

这道题目的思路其实很简单，主要分为两个步骤：

  • 记录每个字符每个顺位出现的次数
  • 排序，按降序提取字符

首先是第一部分，记录每个字符的每个顺位的出现次数，可以使用vector>来进行表示。

这里，可以直接遍历输入的vector就可以了。

然后，第二步就是排序，这里我们需要了解到sort函数可以直接对vector>排序。所以，我们直接排序就可以了。

不过，有一个问题。

排序后的index会发生变化，如何保存其对应的字符呢？

这个时候有两种方法：

  • map
  • vector

使用map的话，相当于构建数组和字符的映射，有点麻烦。

而使用vector的话，可以多开一位，用以记录字符。

这样的话，sort后，可以直接搜寻标记位，就可以直接提取出对应的字符信息了。因此，我们就可以得到投票后的字符信息了。
