思路

矩形重叠

矩形重叠的思路其实很简单，我们将不重叠的情况枚举一下，然后剩下的情况就是重叠的情况。

两个矩形给的都是对角点，所以，我们只需要比较四个坐标就可以了。

我们通过$x-$轴和$y-$轴来进行比较：

  • $x-$轴

我们令两个矩阵的$x-$轴上四个点的坐标为{$A,B,C,D$}，其中$A< B$属于一个矩形，$C < D$属于另一个矩形。

那么我们可以知道，不重叠的情况为两个矩阵的$x-$轴无交集。

故，可以得到两种情况：

$B < C \quad ||\quad D < A$

这两种情况是矩阵不相交的，同时，题目要求边相交也不算重叠，那么我们就把等号加进去，变为：

$B \leq C \quad ||\quad D \leq A$

这样，就得到了不重叠的$x-$轴的判定条件

  • $y-$轴

$y-$轴也是同样的道理，我们可以通过相似的思路来得到$y-$轴上的四个坐标：{$E,F,G,H$}，其中$E < F$属于一个矩形，$G < H$属于另一个矩形。

通过对其判定，我们可以得到不重叠的情况为：

$E \leq F \quad || \quad G \leq H$

所以，最终，我们可以得到重叠的判定条件为：

$B \leq C \quad ||\quad D \leq A \quad || \quad E \leq F \quad || \quad G \leq H$

那么不重叠的判定条件就为：

!($B \leq C \quad ||\quad D \leq A \quad || \quad E \leq F \quad || \quad G \leq H$)

矩形面积

矩形面积这道题目其实是承接上面的这道题目的。

我们设定，第一个矩形的面积为$Area_A$， 第二个矩形的面积为$Area_B$，重叠区域的面积为$Area_D$。

那么这道题目就可以分为两种情况：

- 两个矩形有重叠区域，面积为：$Area_A + Area_B - Area_D \quad \& \quad Area_D > 0$

- 两个矩形无重叠区域，面积为：$Area_A + Area_B$

从题目中可以知道：

$Area_A = (C-A)(D-B) \quad \& \quad Area_B = (G-E)(H-F)$

我们可以只看一个轴，$x$轴：

从$x$轴来看，重叠的时候一共可以分为四种情况：

  • AC在左，AC不包含EG，按顺序排列为：[A,E,C,G];

  • AC在左，AC包含EG， 按顺序排序为：[A,E,G,C];

  • EG在左，EG不包含AC，按顺序排序为：[E,A,G,C];

  • EG在左，EG包含AC， 按顺序排序为：[E,A,C,G].

所以，上面四种情况就可以确定了。我们需要的是中间的重叠部分，那么其实计算起来很简单：

两条线段总长 - 最长的线段。

所以，就可以求解这道题目了。

这道题目在具体实施的时候，还有一个小细节需要注意：

就是最后求解的时候，不要使用$Area_A + Area_B - Area_D$，最好使用$Area_B - Area_D + Area_A$，这样可以防止算出来的值超过int型，造成执行错误。
