3.21-水壶问题

发表于 2020-03-21 | 分类于 LeetCode ， 每日一题

LeetCode365. 水壶问题

贝祖定理

题目

思路

这道题目其实是一个数学逻辑思维的题目。

我们其实可以列出一个数学公式：

$ax + by = z, \quad s.t \quad a \in N, b \in N$

所以，我们就可以继续后面的计算。

由于水壶之间之内互相倒水，装满或倒空。所以，可以得到的第一个条件就是：

$x + y \geq z$

如果不满足这个条件，则两个水壶装满了也不能满足想要的水量，所以，此时肯定不可能。

然后，剩下的就是依赖贝祖定理了。

我们假设$x,y$的公约数为$t$.

则$ax + by = aa_1t + bb_1t = (aa_1 + bb_1)*t$

那么其实这个时候，问题就变得简单了:

$aa_1 + bb_1 \in N^+$

所以，$t$应该是需要可以被$z$整除的。

即：$z \% t = 0$

所以，我们只需要算出$x$和$y$的公约数，然后判断是否可以被整除就可以了。
