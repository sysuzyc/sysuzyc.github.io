思路

这道题目，其实思路还是比较清晰。我们需要找到的就是能够后缀完全匹配的单词即可。

  • 可以按照字符串的逆序进行排序，这样，后缀一样的单词会靠近再一起
  • 判定后缀是否完全贴合，如果完全贴合的话，就可以忽略
  • 最后，计算还剩下多少单词，记得加上”#”就完成这道题了
