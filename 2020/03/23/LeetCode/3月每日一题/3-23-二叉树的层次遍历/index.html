思路

上面的两道题其实很接近。我们先解决第一个问题：

  • 二叉树的层次遍历

我们知道，树的遍历一般使用第归来求解，所以，我们就用递归的方法来书写。

因为是层次遍历，所以，一定要有层数的概念。

加入了层的概念后，就可以很好的完成递归了。

所以，这个时候，就需要根据传入的层数来判断。

在对应的层数存入相应的数据，然后层数加1，迭代子节点。

这样就可以完成二叉树的层次遍历了。

  • 二叉树的锯齿形层次遍历

这道题目和前面稍微有点不同。

二叉树的遍历根据层的奇偶性来进行不同的存储顺序的变动。

这个时候就涉及到了vector的存值的性质了。

一般来说都是push_back，直接往后压制。

但是，当锯齿遍历后，其实需要的是往前压制。

这个时候，直接insert(vec.begin(),value)，就可以了。
